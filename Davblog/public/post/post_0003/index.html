测试更新完毕。

大概汇报下近况以及今年的计划。

可以说是自从19年回家之后直到现在的时间，都让我觉得十分漫长。自身习惯了懒散的生活节奏之后，再想回到一个学生的状态真的很难。在无数遍调整心态之后，终于有了些“正在上学”的感觉。

从今年4月开始的春季学期，我作为履修生（也就是旁听生/预科生）开始在早稻田的鷲崎（Washizaki）研究室学习。鷲崎教授的研究方向主要是Software Engineering相关，而我选择的研究方向是Programming Education，是教授研究方向中的一个小小的分支。如果顺利的话，应该是今年10月份开始申请早稻田的AO项目（也就是英文生），明年4月的春季学期才能作为一名正式的修士在早稻田学习。

到明年4月为止还有一年的时间，我的个人计划姑且有以下几方面：

  • 找到研究方向，并在10月前做出一份研究报告。
  • 重新恢复更新网站，更新内容以每周学习笔记为主。
  • 每周尽可能读2-3篇论文，读论文速度上来了之后再加大数量。
  • 每周完成课程内容，尽可能期末得个好点的成绩。

这学期选的课程是4节：

  • Software quality assurance
  • Pattern Recognition and Machine Learning
  • Foundations for Information Access Evaluation
  • 高信頼ソフトウェア

目前感觉有难度的是机器学习这节课，因为需要用到线性代数的知识。比如说第二节课的作业的一道题，就需要根据一些2D样本，计算出特征向量（PCA里面的PC1），另一道题则是要用LDA算出2组2维样本的一维子空间（特征向量）。这两道题中都涉及到了我没有学过的矩阵计算、协方差矩阵的概念啥的。总之算出那2道题的过程十分艰辛，而且最后我也不确定我算没算对（笑）…

（说到这个，强推结城浩的《程序员的数学1-3》，简单易懂，救了我的命）

其余的课程暂时都还好。虽然也涉及到了很多新知识，但总体来说问题应该不大。

想说的大概就是这些。

最近在思考的问题是“少儿编程教育”有什么方向可以写论文或做研究。目前的项目组的教授是做 少儿编程教育+游戏开发的，于是自己最近也在研究游戏开发的基础。
